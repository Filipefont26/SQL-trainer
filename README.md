Este projeto consiste no desenvolvimento e deploy de uma aplicação web para treino de SQL, disponibilizada publicamente via cloud.

👉 A aplicação está online:
🔗 https://sqltrainner-554807962036.europe-west1.run.app/

O principal objetivo foi simular um cenário real de produto, indo além da análise de dados e explorando deploy, infraestrutura e disponibilização de aplicações na nuvem.

🎯 Objetivo

Criar uma aplicação que permita:

Praticar consultas SQL de forma interativa
Simular um ambiente de aprendizado prático
Disponibilizar o sistema de forma acessível via web
Aplicar conceitos reais de deploy em cloud
🛠️ Tecnologias Utilizadas
Python
SQL
Streamlit
Google Cloud Platform (GCP)
Cloud Run (deploy serverless)
☁️ Arquitetura do Projeto

A aplicação foi construída e publicada utilizando o Cloud Run, serviço serverless do GCP.

👉 O Cloud Run permite rodar aplicações em containers sem necessidade de gerenciar servidores, com escalabilidade automática e alta disponibilidade

🔧 Fluxo da aplicação:
Aplicação desenvolvida em Python (Streamlit)
Containerização da aplicação (Docker)
Deploy no Cloud Run
Acesso público via URL
🚀 Funcionalidades
Interface simples para treino de SQL
Execução de queries
Simulação de ambiente real de análise
Acesso direto via navegador (sem instalação)
💡 Problema Resolvido

Antes:

Necessidade de ambiente local para treinar SQL
Configuração manual de banco e ferramentas
Baixa acessibilidade

Depois:

Aplicação online e acessível de qualquer lugar
Ambiente pronto para uso
Facilidade para praticar SQL rapidamente
🔥 Diferencial do Projeto

Este projeto se destaca por ir além da análise de dados:

Deploy em ambiente cloud (produção real)
Uso de arquitetura serverless
Aplicação acessível publicamente
Integração entre desenvolvimento e infraestrutura

👉 Isso demonstra não só habilidade analítica, mas também conhecimento em engenharia de dados e cloud
